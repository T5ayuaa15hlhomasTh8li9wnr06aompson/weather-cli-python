
# Weather CLI - Python

一个简单的天气查询脚本，使用 OpenWeather API，支持中文城市名。

## 使用方法

1. 安装依赖：

```bash
pip install requests
```

2. 运行脚本：

```bash
python weather.py
```

3. 输入城市名后即可显示天气。

## 注意事项

- 你需要先去 [https://openweathermap.org/](https://openweathermap.org/) 申请一个 API Key，并填入代码中。
